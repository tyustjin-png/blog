1. MBPпјҡзәҜеҮҖз»Ҳз«Ҝ#

еҺҹеҲҷпјҡдё»еҠӣз¬”и®°жң¬дёҚиЈ…д»»дҪ•еӨ§жЁЎеһӢзӣёе…ізҡ„е№іеҸ°жҲ–зі»з»ҹгҖӮ

MBPеҸӘеҒҡдёүд»¶дәӢпјҡ

  • VSCode + SSHпјҡиҝңзЁӢиҝһMac MiniеҶҷд»Јз ҒгҖҒз”ЁClaude Code
  • Telegramпјҡи·ҹзҙ«йҫҷе’ҢдәҢзӢ—еҜ№иҜқ
  • жөҸи§ҲеҷЁпјҡзңӢеҚҡе®ўгҖҒз®ЎзҗҶCloudflare

дёәд»Җд№Ҳпјҹеӣ дёәеӨ§жЁЎеһӢе№іеҸ°иЈ…еңЁжң¬ең°дјҡеёҰжқҘи®ӨиҜҒгҖҒжӣҙж–°гҖҒиө„жәҗеҚ з”ЁзӯүдёҖе Ҷй—®йўҳгҖӮжҠҠз®—еҠӣе’ҢжңҚеҠЎе…ЁйғЁйӣҶдёӯеңЁMac MiniдёҠпјҢMBPдҝқжҢҒиҪ»йҮҸе№ІеҮҖгҖӮ

2. Mac Mini M4пјҡдёҖеҲҮзҡ„еј•ж“Һ#

иҝҷжҳҜж•ҙеҘ—зі»з»ҹзҡ„ж ёеҝғгҖӮ7Г—24е°Ҹж—¶ејҖжңәпјҢи·‘зқҖжүҖжңүжңҚеҠЎпјҡ

OpenClaw вҖ” AIеҠ©жүӢжЎҶжһ¶

  • зҙ«йҫҷпјҲдё»Agentпјүпјҡз”ЁClaude Opus 4.6пјҢеӨ„зҗҶж—ҘеёёжІҹйҖҡгҖҒеҶҷдҪңгҖҒзі»з»ҹз®ЎзҗҶ
  • дәҢзӢ—пјҲжҠ•иө„Agentпјүпјҡдё“й—ЁеҒҡжҠ•иө„еҲҶжһҗпјҢи·ҹзҙ«йҫҷдә’дёҚе№Іжү°
  • йҖҡиҝҮTelegramи·ҹжҲ‘еҸҢеҗ‘йҖҡдҝЎ

Claude Code вҖ” зј–зЁӢеҲ©еҷЁ

  • з”Ёsub2apiз”ҹжҲҗзҡ„API Keyи®ӨиҜҒпјҢдёҚйңҖиҰҒOAuthзҷ»еҪ•
  • д»ҺMBPзҡ„VSCode SSHиҝҮжқҘзӣҙжҺҘиғҪз”Ё
  • и·ҹOpenClawе…ұз”ЁеҗҢдёҖдёӘAPI KeyпјҢдә’дёҚеҶІзӘҒ

Chrome Debug вҖ” зҪ‘йЎөж•°жҚ®жҠ“еҸ–

  • з«ҜеҸЈ9222пјҢCDPеҚҸи®®
  • дё“з”Ёж•°жҚ®зӣ®еҪ•пјҢи·ҹж—ҘеёёChromeдёҚеҶІзӘҒ
  • launchdејҖжңәиҮӘеҗҜ

1Password CLI вҖ” еҜҶй’Ҙз®ЎзҗҶ

  • Service AccountжЁЎејҸпјҢеҸӘиғҪи®ҝй—®Agent Vault
  • API KeyгҖҒеҜҶз Ғзӯүж•Ҹж„ҹдҝЎжҒҜиҝҗиЎҢж—¶иҜ»еҸ–пјҢз”Ёе®ҢеҚіејғ
  • йӣ¶жҳҺж–ҮеӯҳеӮЁ

3. и…ҫи®Ҝдә‘VPSпјҡе®үе…Ёжү§иЎҢеҷЁ#

е®ҡжҠ•и„ҡжң¬и·‘еңЁVPSдёҠпјҢзӣҙиҝһеёҒе®үAPIгҖӮдёәд»Җд№ҲдёҚеңЁMac MiniдёҠи·‘пјҹ

е®үе…Ёйҡ”зҰ»пјҡAPI KeyеңЁVPSжң¬ең°зҡ„1PasswordйҮҢпјҢйҖҡиҝҮеҶ…еӯҳиҜ»еҸ–пјҢдёҚиҝҮзҪ‘з»ңдј иҫ“гҖӮеҚідҪҝMac Miniиў«е…ҘдҫөпјҢд№ҹжӢҝдёҚеҲ°дәӨжҳ“еҜҶй’ҘгҖӮ

зҪ‘з»ңеҠ еӣәпјҡ

  • SSHз«ҜеҸЈж”№дёә52222пјҲйқһй»ҳи®Ө22пјү
  • Fail2Banпјҡ3ж¬Ўй”ҷиҜҜе°ҒзҰҒ1е°Ҹж—¶
  • д»…TailscaleеҶ…зҪ‘еҸҜиҝһжҺҘпјҢе…¬зҪ‘йӣ¶з«ҜеҸЈжҡҙйңІ

4. Tailscaleпјҡз»„зҪ‘иғ¶ж°ҙ#

Mac Miniе’ҢVPSйҖҡиҝҮTailscaleз»„жҲҗз§ҒжңүзҪ‘з»ңпјҡ

  • Mac Miniпјҡ100.99.51.77
  • VPSпјҡ100.72.82.26

жүҖжңүи®ҫеӨҮй—ҙзҡ„йҖҡдҝЎйғҪиө°TailscaleеҠ еҜҶйҡ§йҒ“пјҢдёҚжҡҙйңІе…¬зҪ‘з«ҜеҸЈгҖӮ
